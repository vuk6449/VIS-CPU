## VIS CPU
A fun project made in the free time of Vuk and Steven.

Build with either:
- ```make```
- ```make build```

Run the project with either:
- ```make run ARGS="file.bin"``` (will choose file.bin)
- ```make run``` (will choose main.bin)

Clean the project with: ```make clean```
